# HackerRank — React

Challenges focused on React fundamentals and component thinking, and styling — including CSS and Styled Components.

The challenges can be found in [react-exercises/src/exercises](./react-exercises/src/exercises/), each containing its own separate folder.

## Solutions

| # | Challenge | Concepts | Difficulty | Import on main.tsx | 
|---|-----------|----------|------------|--------------------|
| 1 | [Item List Manager](./react-exercises/src/exercises/1_item-list-manager/) | controlled input, useState | Medium, Basic | `import App from './exercises/1_item-list-manager/App.tsx'` | 
| 2 | [Code Review Feedback](./react-exercises/src/exercises/2_code-review-feedback/) | useState, Styled Components, CSS Grid, dark mode | Medium, Advanced | `import App from './exercises/2_code-review-feedback/App.tsx'`| 

## Concepts covered so far

- controlled input
- useState (array e string)
- updater function pattern
- imutabilidade com spread
- `useEffect` pra side effects fora do ciclo de render
- Styled Components (props dinâmicas, transient props, GlobalStyle)
- CSS Grid com grid-template-areas
- CSS variables pra theming (dark/light mode)
- animação com @keyframes + trick da key pra replay
